# DockStart Basic Mode Demo

This is a tiny DockStart demo project for software workflow testing only.

It contains toy `prepared/receptor.pdbqt` and `prepared/ligand.pdbqt` files so a new user can run the Basic Mode path when AutoDock Vina is available:

```text
已有 PDBQT -> 设置 Box -> 生成配置 -> 运行 Vina -> 解析结果 -> 导出报告
```

The structures are intentionally minimal and are **not suitable for scientific conclusions, docking interpretation, drug efficacy claims, or publication**.
