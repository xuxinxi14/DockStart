# DockStart Assisted Mode Demo

This is a tiny DockStart demo project for software workflow testing only.

It contains toy raw receptor/ligand files so a new user can see the Assisted Mode path:

```text
raw PDB/SDF -> Python + RDKit + Meeko -> prepared PDBQT
```

The files are deliberately small and are **not suitable for scientific conclusions, docking interpretation, drug efficacy claims, or publication**.
